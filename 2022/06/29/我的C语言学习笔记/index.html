位操作

计算机对于数据的处理都是以补码的形式进行的

位操作符

符号 描述 运算规则
& 按位与 两个位都为1时，结果才为1
\ 按位或 两个位都为0时，结果才为0
^ 按位异或 两个位相同为0，相异为1
~ 按位取反 0变1，1变0
<< 左移 各二进位全部左移若干位，高位丢弃，低位补0
>> 右移 各二进位全部右移若干位。无符号数，高位补0；有符号数，各编译器处理方法不一样，有的补符号位（算术右移），有的补0（逻辑右移）

注意事项

  1. 只有~取反是单目操作符，其他都是双目操作符
  2. 位操作只能用于整型数据
  3. 位操作符的运算优先级比较低，因此尽量使用括号来确保运算顺序
  4. 有符号数负数的右移，不同系统的处理方式不一样，有高位补1的，也有高位补0的
  5. 复合操作符，&=、|=、^=、<<=、>>=。

typedef int Sint;
//构造顺序栈的结构体
typedef struct Slist
{
Sint base; //栈底指针
Sint *top; //栈顶指针
int scapacity; //栈容量
} Slist;

//宏定义一个栈容量

define Slist_max_size 100;

//初始化栈
void
InitSlist(Slist list)
{
list->base = (Sint )malloc(sizeof(Sint));
if (!list->base)
exit(0); //分配失败
else
{
list->top = list->base;
list->scapacity = Slist_max_size;
printf(“初始化成功\n”);
}
}


//进栈
void push(Slist list, Sint data)
{
//判栈满
if (list->top == list->base + list->scapacity)
{
printf(“栈满了！\n”);
}
else
{ (list->top) = data;
list->top++;
}
}

//出栈
Sint pop(Slist list)
{
//判栈空
if (list->top == list->base)
{
printf(“空栈！\n”);
return 0;
}
else
{
list->top—;
return (list->top);
}
}
